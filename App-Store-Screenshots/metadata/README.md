## Localization

Currently only `en-US` is configured. Files follow fastlane metadata conventions:

- `name.txt` — App name
- `subtitle.txt` — Subtitle
- `keywords.txt` — Keywords (max 100 chars)
- `description.txt` — Full description
- `promotional_text.txt` — Promotional text (can be updated without new build)
